Find Magic Items

The story behind this project is a fun one! I was playing Dungeons and Dragons a little bit ago, and our Game Master complained about how they always had to come up with treasure for us by hand. I took that to heart, found a list of magic items that existed in the game on the internet, and created this webpage! I originally created it using PostgreSQL and Python3. I changed to PostgreSQL and PHP for ease of use with Heroku. I have since posted this in forums for Game Masters to use around the world and it has been well recieved.

This Website

This website was made using HTML and Git Pages, a service I didn't know existed before I began working on this project. I wanted a site that I could keep track of and show off my projects easily, and Git Pages was a fun learning experience.
